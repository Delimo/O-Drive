### 手动验收清单

- [ ] ZIP 下载：单文件夹递归打包下载 — 选中文件夹 → 点"批量下载" → 下载ZIP
- [ ] ZIP 下载：批量选中多文件/多文件夹混合下载 — 选中2+项 → 点"批量下载" → 下载ZIP
- [ ] 视频预览：能打开、能拖动进度条 — 点击视频文件预览 → 拖动进度条
- [ ] 音频预览：能打开、能拖动进度条 — 点击音频文件预览 → 拖动进度条
- [ ] 拖拽上传：拖入单个文件 → 弹出确认弹窗 → 上传
- [ ] 拖拽上传：拖入文件夹 → 递归展开所有文件 → 确认弹窗 → 上传
- [ ] 拖拽上传：拖入多个文件/文件夹混合 → 确认弹窗 → 上传
- [ ] 断点续传：大文件上传中断后刷新页面 → 重新选择同一文件 → 自动跳过已完成分片
- [ ] 断点续传：上传完成后 localStorage 中对应条目自动删除
