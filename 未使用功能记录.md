|------|------|

## 2. 基于保留天数的回收站自动清理

| 项目 | 内容 |
|------|------|
| **端点** | `POST /api/trash/cleanup` |
| **后端位置** | `functions/api/lib/trash.js:375` — `handleTrashCleanup` |
| **路由注册** | `functions/api/lib/router.js:217-218` |
| **功能** | 读取 `trash_retention_days` 设置，永久删除超过保留天数的回收站项目 |
| **前端缺失** | 没有触发入口（前端只有"清空全部"和单条删除，没有按保留策略清理的按钮或定时任务） |

## 建议补 UI 位置
