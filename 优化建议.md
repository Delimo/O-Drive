## P1 — 高优先级 ✓

### 2. `TOKEN_SECRET` 空值降级加固

- **文件**: `functions/api/lib/secrets.js:20-31`
- **变更**:
  - 移除 `typeof process !== "undefined"` 判断，使警告在 Cloudflare Workers 环境下也能输出
  - 控制台警告始终触发（不再因 `NODE_ENV=test` 跳过）
  - 生产环境在日志中可追踪到"使用了不安全默认密钥"的警告

### 3. 搜索缓存加大小上限

- **文件**: `functions/api/lib/file-index.js:557-558`
- **变更**:
  - 新增 `SEARCH_CACHE_MAX_SIZE = 500` 常量
  - 设置新缓存条目时，若超过 500 条则淘汰最早条目（Map 自然顺序）
  - 防止查询多样化时内存无限制增长

### 4. HMAC 签名密钥缓存加 TTL 和大小上限

- **文件**: `functions/api/lib/s3-signing.js:1`
- **变更**:
  - 新增 `KEY_CACHE_TTL = 3600000`（1 小时 TTL）
  - 新增 `KEY_CACHE_MAX_SIZE = 200`
  - 获取缓存时校验 TTL，过期重新计算
  - 设置新条目时超过上限则淘汰最早条目
