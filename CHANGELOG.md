# 更新日志 (CHANGELOG)

## [V1.0.0] - 2026-05-08
### 🚀 新增功能
- **重构架构**：从单一 Worker 架构迁移至 Cloudflare Pages + 模块化 CSS/JS。
- **多视图切换**：支持“网格视图”和“列表视图”，并增加“修改时间”展示。
- **上传管理系统**：引入带进度条的右下角任务面板，支持多文件并发上传及自动隐藏。
- **文件剪贴板**：实现全局“移动/复制/粘贴”逻辑，支持文件夹递归迁移。
- **在线编辑器**：管理员可直接在线编辑纯文本和代码类文件。
- **搜索增强**：支持根据当前路径自动切换“全盘搜索”或“子目录搜索”。
- **隐私盾**：增加隐私路径管理功能，名单内路径对访客完全不可见。
- **全局通知**：新增 Toast 气泡提示系统，所有操作均有即时反馈。

### 🎨 UI/UX 优化
- **现代深色系**：全站采用 Indigo / Navy 风格。
- **沉浸式布局**：固定顶栏与工具栏，实现主文件区独立滚动，美化滚动条样式。
- **响应式对齐**：Header 元素向中心靠拢对齐主容器，列表视图严格列对齐。
- **登录交互**：将独立登录页改为 Element Plus 风格的毛玻璃弹窗。

### 🛡️ 安全增强
- **分级鉴权**：实现访客（只读）与管理员（全权）权限彻底隔离。
- **登录加固**：增加管理员用户名校验及 IP 暴力破解频率限制。
- **路径审计**：每一次上传、删除、重命名、编辑均记录在 D1 日志表中。

### 🐛 修复
- 修复了根目录出现空名称“幽灵文件夹”的问题。
- 修复了重命名时光标无法点击定位的问题。
- 修复了文件后缀名在重命名时被误选的问题。
- 修复了多次点击保存或回车导致的重复提示 Bug。
