# 更新日志

项目首次正式发布前使用 `0.x` 版本。后续发布应同步更新本文件，并在公开版本稳定后创建对应 Git Tag。

## 0.1.0（开发中）

- 优化 API 只读请求限流，保留本地逐请求限制并降低 D1 写入频率。
- 使用 D1 batch 合并核心建表与索引语句，减少冷启动往返。
- 新增分享页文件、密码解锁和文件夹浏览的 Playwright 测试。
- 集中注册前端 `data-action`，检测 action 重名并移除广播式点击分发。
- 删除 CSP 阻断的内联事件，增加图片加载失败的统一回退处理。
- 改进自定义选择器键盘操作、弹窗焦点管理和 Toast 无障碍语义。
- 补齐自定义日期选择器键盘操作、筛选控件名称、表单标签和存储进度条的读屏语义。
- 修复自定义选择器 Escape 冒泡导致弹窗被同时关闭的问题。
- 修复 Windows 环境下 Playwright 测试服务器无法正常退出的问题。
- 修复 R2 存储配额切换 GB/MB 后表单仍提交旧单位的问题。
- 修正文档中的审计状态、测试数量和已知行为说明。
- Cloudflare R2 内容去重、文件索引和存储对象引用计数。
- 文件与文件夹分享、密码访问、ZIP 下载和过期分享重新启用。
- 回收站、后台任务、通知中心、Webhook、WebDAV 和维护工具。
- 首页、管理后台和分享页的模块化前端架构与自动化测试。
