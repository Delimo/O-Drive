# Changelog

本项目只记录较大的结构、功能和部署相关改动。细碎样式微调和小修复不单独列入。

## 2026-05-10

### Changed

- 将原本集中在 `index.html` 和 `admin.html` 中的大段脚本拆分为独立前端模块，入口改为 `public/js/app.js` 和 `public/js/admin-app.js`。
- 将 Cloudflare Pages Functions 后端拆分为路由、鉴权、管理、文件读取、文件写入等模块，降低单文件维护压力。
- 优化 Cloudflare Pages 部署说明，补充构建命令、输出目录、D1/R2 绑定和环境变量说明。
- 重新整理文件浏览器界面，包括文件卡片、列表视图、操作按钮、预览弹窗和整体暗色配色。

### Fixed

- 修复预览弹窗被当作普通页面块显示在底部的问题。
- 修复根目录文件列表中可能出现空文件夹项的问题。
- 修复文件卡片中“预览 / 下载”和文件大小显示互相挤压的问题。
